# Antigravity 2.0 프로그램 환경 설정 및 확대 가이드

본 문서는 현재 사용 중이신 **Antigravity 2.0 데스크톱 클라이언트(채팅 및 AI 에이전트 도구)**의 화면 글씨 크기를 키우는 방법과 설정 내용을 한글로 정리한 가이드입니다.

---

## 🔍 1. Antigravity 2.0 화면 및 글씨 크기 키우기 (가장 확실한 방법)

현재 사용하고 계신 Antigravity 2.0 프로그램은 Chromium/WebView2 기반의 데스크톱 애플리케이션으로, 인터넷 브라우저나 Discord, Slack 등과 동일한 **표준 단축키**를 사용하여 화면 전체 배율을 즉시 조절할 수 있습니다.

### 💡 화면 글씨 및 모든 요소 확대 (단축키)
*   **화면 확대 (Zoom In):** **`Ctrl` + `+`** (또는 **`Ctrl` + `=`**)
    *   이 단축키를 누르면 채팅 텍스트, 사이드바, 아이콘 등 화면 전체가 한 단계씩 커져 가독성이 향상됩니다.
*   **화면 축소 (Zoom Out):** **`Ctrl` + `-`**
*   **원래 기본 크기로 초기화 (Reset):** **`Ctrl` + `0`**
*   **마우스 휠 조절:** **`Ctrl` 키를 누른 상태에서 마우스 휠을 위로 굴리면** 화면이 확대됩니다.

## ⚙️ 2. IDE 개발 도구 내 글씨 크기 설정 (settings.json)

개발 소스 코드를 보는 에디터 창의 폰트를 키우고 싶으실 때는 아래 설정을 편집하실 수 있습니다.

*   **설정 파일 경로:** `C:\Users\tvxqc\AppData\Roaming\Antigravity IDE\User\settings.json`

| 설정 키 (Settings Key) | 추천 설정 값 | 한글 설명 |
| :--- | :--- | :--- |
| `editor.fontSize` | `18` | **에디터 본문의 글씨 크기**입니다. (18~20 권장) |
| `window.zoomLevel` | `0.5` | **에디터 프로그램 전체 배율(줌 레벨)**입니다. 0.5~1.0 사이로 설정하면 전체 메뉴가 커집니다. |
| `git.autofetch` | `true` | 원격 저장소의 최신 변경 내역을 주기적으로 자동 페치(Fetch)합니다. |
